# Как закоммитить изменения

## Шаг 1: Проверь что изменилось

```bash
cd ~/projects/bots/yomi_profit_test
git status
```

---

## Шаг 2: Добавь все файлы

```bash
git add .
```

---

## Шаг 3: Сделай коммит

```bash
git commit -m "Фаза 4 завершена: Mini App + Flask API

- Flask API сервер на порту 8001
- Многостраничная архитектура Mini App
- Создание проектов, профессий, услуг
- Динамические прогрессивные ставки
- Навигация БЕЗ закрытий
- Использование fetch() вместо sendData()
- Обновлена документация"
```

---

## Шаг 4: Запуш в GitHub

```bash
git push origin main
```

---

Готово! ✅
